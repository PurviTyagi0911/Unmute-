# FEATURES

## Student Features

* View campus events
* Search events
* Filter by category
* Filter by date
* View event details
* View announcements
* Browse events without registration
* Notification about every new post

---

## Club Features

* Club login
* Create events
* Edit events
* Delete events
* Upload event posters
* Post announcements

---

## Event Information

Each event contains:

* Title
* Category
* Date
* Time
* Location
* Description
* Poster Image
